# Remove-QualysAssetGroups

## SYNOPSIS
Removes an Asset Group to Qualys

## SYNTAX

```
Remove-QualysAssetGroups [-Identity] <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Removes an Asset Group to Qualys

## EXAMPLES

### EXAMPLE 1
```
Remove-QualysAssetGroups -Identity "My Asset Group"
```
